# Ciron

Ciron includes more than 8.7k posts, collected from the Weibo, a microblogging platform. Ciron is one scale larger in size than the previously available dataset. Ciron is collected with no pre-conditions to ensure much wide coverage. Five fine-grained classes for ironic ratings:1 (not ironic), 2 (unlikely ironic), 3 (insufficient evidence), 4 (weakly ironic), 5 (strongly ironic).
